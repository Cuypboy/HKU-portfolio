# HKU Portfolio

Een minimale Next.js-site voor het HKU-toelatingsportfolio van Joep.

## Lokale ontwikkeling

```bash
npm install
npm run dev
```

## Deployment

Dit project is bedoeld voor GitHub en Vercel. Zodra de repository aan Vercel is gekoppeld, kan Vercel automatisch deployen na iedere push naar GitHub.
